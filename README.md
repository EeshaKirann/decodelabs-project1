# Software Agency Landing Page

A fully responsive, modern software agency landing page built with HTML5, CSS3, and JavaScript.  
It is designed to showcase a company's services, portfolio, and team in a clean and professional interface.

## Overview

This project presents a polished landing page for a software agency with a mobile-first layout, smooth interactions, and strong visual structure.  
It focuses on accessibility, responsive design, and a user-friendly experience across all screen sizes.

## Features

- Fully responsive, mobile-first layout.
- Semantic HTML5 structure for better accessibility.
- CSS Grid and Flexbox-based design system.
- Animated scroll-reveal sections.
- Live-counting statistics.
- Interactive contact form with validation.
- Mobile navigation menu.
- Smooth scrolling and a Back to Top button.
- Active navigation highlighting.

## Technologies Used

- HTML5
- CSS3
- JavaScript
